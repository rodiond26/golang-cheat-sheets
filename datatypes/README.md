#### Целые числа

* Знаковые:
    * int, int8, int16, int32, int64
* Беззнаковые:
    * uint, uint8, int16, uint32, uint64
* Символы unicode:
    * type rune = int32
* Байты:
    * type byte = int8

#### Числа с плавающей точкой

* float32 (точность ~6 знаков), float64 (точность ~15 знаков)
* complex64, complex128

#### Логический тип:

* bool:
    * true, false
